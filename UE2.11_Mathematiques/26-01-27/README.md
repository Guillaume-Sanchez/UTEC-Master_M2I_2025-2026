Légende :
```
⊂ -> inclu
∈ -> Appartient
∀x -> Pour tout x
Ø -> Enssemble vide
! -> factoriel
```

E = {a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z}

F = {a, b, c, d, e} 

F ⊂ E : ∀x ∈ F , x ∈ E

F ⊂ E

>Propriété : 
>
>Ø ⊂ E 
>
>E ⊂ E

## Exo 1

Soit E = {+,-,*,:}
écrire P(E), l'ensemble des parties de E

P(E) = {Ø, {+}, {-}, { * }, {:}, {+,-}, {+, * }, {+,:}, {-, * }, {-, :}, { *, :}, {+, -, * }, {+, -, :}, {+, * , :}, {-, * , :}, E}

Donc cardinal de P(E) = 2⁴ = 16

Il y a donc 16 sous enssemble.
